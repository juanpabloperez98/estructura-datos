Arreglos

¿Qué es un arreglo?

Los arreglos son otra estructura de datos estática, pero con mayor beneficio que las variables simples. La gran mayoría de soluciones por computadora no será suficiente con el uso de variables simples, necesitará de una colección de datos del mismo tipo (arreglos) o diferentes tipos(estructuras), incluso archivos. Los arreglos permiten manipular datos de manera muy flexible. Combinándolas y anidándolas, es posible organizar información de manera estructurada para representar sistemas del mundo real.

Un arreglo es una colección de posiciones de almacenamiento de datos, donde cada una tiene el mismo tipo de dato y el mismo nombre. Cada posición de almacenamiento en un arreglo es llamada un elemento del arreglo.

Declaración

Un arreglo es una colección de posiciones de almacenamiento de datos, donde cada una tiene el mismo tipo de dato y el mismo nombre. Cada posición de almacenamiento en un arreglo es llamada un elemento del arreglo.

  • Escribe el tipo de dato que almacenará el arreglo
  • Escribe el nombre del arreglo
  • Entre corchetes, escribe la cantidad de elementos de ese tipo que se almacenarán.

                    float grupo[10];

    Asignar valores a una ubicación

    Los arreglos nos sirven para almacenar datos. Si deseamos guardar datos en un arreglo podemos escribir la ubicación exacta donde deseamos almacenarlos.

    Para guardar un dato en este caso, un cero- en las ubicaciones 0,1 y 2 del arreglo grupo, podemos escribir: 

                    
                    grupo[0] = 0;
      
grupo[1] = 0;

grupo[2] = 0;
