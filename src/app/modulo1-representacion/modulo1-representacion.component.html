Representacion binaria

La representación binaria de los datos se usa en el lenguaje máquina, todos los tipos de datos tienen su representación binaria por lo que cada tipo de dato tiene un tamaño de espacio en la memoria de la máquina.

Los tipos de datos determinan el espacio de almacenamiento que se debe asignar junto con los tipos de operaciones permitidas.

Un ejemplo de cómo se almacenan los enteros de forma binaria:

Numero = 13;

Representación decimal: 13 = 1*10^1 + 3*10^0

Representación binaria: 1101 = 1 * 2^3 + 1 * 2^2 + 0 * 2^1 + 1 * 2^0 ModalExample

Cada 1 o 0 es llamado bit, por lo que el numero 13 requiere de 4 bits de almacenamiento.

Para representar –127 en forma binaria se procede de la siguiente manera:

  • Se convierte el 127 a binario: 127 = 111 1111.

  • Como es negativo se representa cada bit de la siguiente manera: se pone un 0 si hay un 1 y si hay un 0 se pone un 1. De modo que se tiene 000 0000.

  • Se toma su signo, en este caso como es negativo se le agrega de complemento un 1 entonces queda: 10000000
