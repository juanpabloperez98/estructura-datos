Colas con prioridades

La cola de prioridad también es un tipo de datos abstracto. Cada elemento en la cola de prioridad tiene su propia prioridad. De hecho, este concepto se entenderá un poco más rápido haciendo algunas comparaciones. Por ejemplo, el sistema de Windows que todos han usado, cuando abrimos el administrador de tareas, la prioridad de cada tarea es diferente, y el sistema operativo seleccionará la tarea con la prioridad más alta para ejecutar primero, y también podemos elegir Marque la prioridad de la tarea.

Otro ejemplo son los médicos de urgencias en turnos nocturnos. Si dos o tres pacientes con resfriados y fiebres están haciendo cola para recibir tratamiento, en este momento, un paciente con un ataque cardíaco tiene prisa. Por supuesto, nuestro médico primero debe tratar el ataque cardíaco. El paciente enfermo La cola de prioridad también es una razón, el procesamiento de prioridad de datos o tareas de alta prioridad.

El elemento con la prioridad más alta se sirve primero, y los elementos con la misma prioridad se sirven según su orden en la cola de prioridad. Las colas de prioridad a menudo se implementan utilizando montones.

La cola de prioridad debe al menos soportar estas operaciones:

  • Insertar elementos prioritarios.
  • Saque el elemento con la más alta prioridad.
  • Ver el elemento de mayor prioridad. (O (1) complejidad de tiempo)
