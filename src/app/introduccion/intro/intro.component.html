Introduccion

¿Qué son las estrucuras de datos?

Las estructuras de datos es la combinación de los tipos de datos básicos, el concepto de tipos de datos viene de la necesidad de representar información y poderla operar en una computadora, para esto es necesario representar esta información de forma binaria.

Codigo ASCII

Los tipos de datos estandarizados vienen de acuerdo con el ANSI (American National Standards Institute) esta pertenece a la Organización Internacional para la Estandarización (ISO). El ASCII (American Standard Code for Information Interchange) que se traduce al español como Código Estándar Estadounidense para el Intercambio de Información, es el código de caracteres basado en el alfabeto latino para el intercambio de información. Este fue creado en 1963 por el Comité Estadounidense de Estándares (ASA), es una evolución de los códigos utilizados en la telegrafía, en 1967, se incluyeron las minúsculas, y se redefinieron algunos códigos de control para formar el código conocido como US-ASCII.

El código ASCII utiliza 7 bits para representar caracteres, fue publicado como estándar en 1967 y su última actualización fue en 1986. Actualmente define códigos para 32 caracteres no imprimibles, de los cuales la mayoría son caracteres de control obsoletos que tienen efecto sobre cómo se procesa el texto, más otros 95 caracteres imprimibles que les siguen en la numeración (empezando por el carácter espacio).

La mayoría de los sistemas informáticos actualmente usan el código ASCII o una extensión compatible para la representación de texto y para el control de periféricos de escritura como el teclado.
