Tipo de datos

¿Qué son los tipos de datos?

Un tipo de dato es un atributo que toma un dato para indicarle a la maquina su comportamiento, los valores que pude tomar y operaciones que puede hacer. Entonces un tipo de dato es un espacio en memoria con restricciones. Por ejemplo, el tipo "int" representa, generalmente, un conjunto de enteros de 32 bits cuyo rango va desde el -2.147.483.648 al 2.147.483.647, también representa las operaciones que se pueden hacer con enteros, como suma, resta, multiplicación y división.

Tipos de datos elementales

Los tipos de datos hacen referencia a la clase de información que se trabaja, donde la unidad mínima de almacenamiento es el dato, también se pude considerar como el rango de valores que pude tomar una variable a lo largo de la ejecución de un programa.

  • Caracteres: El tipo de dato carácter es un dígito individual el cual se puede representar como numéricos (0 al 9), letras (a-z) y símbolos (!"$&/\).

    •                     
                        char caracter;

  • Numéricos: Este tipo de dato puede ser entero o flotante (Real). Los enteros son valores sin punto decimal, pueden ser positivos negativos y cero. Ejemplo: -7, -3, 0, 3, 7. 

    •                     
                        int main(){{ '{' }}
                            printf("Hola mundo");
                        {{ '}' }}

  • Booleanos: Este es tipo de dato se emplea para valores lógicos, su comportamiento se basa en “True” (Verdadero) o “False” (Falso), los podemos como datos comparativos, esta comparación devuelve uno u otro.

    •                     
                        bool valorLogico;

    Tipos de datos compuestos

    Los tipos de datos hacen referencia a la clase de información que se trabaja, donde la unidad mínima de almacenamiento es el dato, también se pude considerar como el rango de valores que pude tomar una variable a lo largo de la ejecución de un programa.

  • Arreglo: almacena un número de elementos del mismo tipo en un orden específico. Los arrays pueden ser estáticos (con una medida fija) o dinámicos (crecer durante su ciclo de vida).

    •                     
                        int my_vector1[10];
                        float my_vector2[25];
                        string my_vector3[500];
                        bool my_vector4[1000];
                        char my_vector5[2];

                        //Formas de declararlo
                        string vector[5] = {{'{'}}"5", "hola", "2.7", "8,9", "adios"{{'}'}};

                        int vector2[] = {{'{'}}1,2,3,4,10,9,80,70,19{{'{'}};

  • Tupla: es un tipo de dato estructurado formado por la unión de varios elementos bajo una misma estructura. Estos elementos pueden ser, o bien datos elementales (entero, real, carácter, etc..), o bien otras estructuras de datos.

    •                     
                        tuple {{'<'}}char, int, float{{'>'}} tupla;

    Ejemplos

    1. Escribir un programa que pregunte al usuario su edad y muestre por pantalla si es mayor de edad o no.
    2. Escribir un programa que almacene la cadena de caracteres contraseña en una variable, pregunte al usuario por la contraseña e imprima por pantalla si la contraseña introducida por el usuario coincide con la guardada en la variable sin tener en cuenta mayúsculas y minúsculas.
    3. Escribir un programa que pida al usuario un número entero y muestre por pantalla si es par o impar.
    4. Para tributar un determinado impuesto se debe ser mayor de 16 años y tener unos ingresos iguales o superiores a 1000 € mensuales. Escribir un programa que pregunte al usuario su edad y sus ingresos mensuales y muestre por pantalla si el usuario tiene que tributar o no.
